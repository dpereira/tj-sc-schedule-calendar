# TJ SC Schedule Calendar

Interactive study schedule calendar for the **GO** cronograma (TJ SC prep) on Gran Cursos Online.

**Period:** May 4 → June 29, 2026
**Events:** 537 (264 May + 273 June), focused on TJ SC (Tribunal de Justiça de Santa Catarina) prep.

## Features

- Month/Week/Day views via [FullCalendar](https://fullcalendar.io/) v6
- Color-coded by subject (12 subjects)
- Completed items shown dimmed (50% opacity)
- Click any event → details modal with direct link to the class on Gran Cursos
- Subject legend with colored dots
- Portuguese locale (pt-br)

## Usage

Open `index.html` in any browser. No server required.

To deploy on GitHub Pages, push to a `gh-pages` branch or use the `docs/` folder.

## Tech

- Single self-contained HTML file (no build step)
- FullCalendar v6 from CDN
- All 537 events embedded inline
